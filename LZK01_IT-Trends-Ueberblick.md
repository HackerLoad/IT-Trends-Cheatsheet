# IT-Trends – Einleitung / Überblick

## 6 wichtige IT-Trends

| Trend | Kurzbeschreibung |
|-------|-----------------|
| Cloud Computing | Bereitstellung von IT-Ressourcen (Server, Speicher, Software) über das Internet auf Abruf |
| Edge Computing | Datenverarbeitung direkt am Rand des Netzwerks, nah an der Datenquelle – nicht in der zentralen Cloud |
| Künstliche Intelligenz (KI) | Systeme, die menschenähnliche Aufgaben automatisiert ausführen (Spracherkennung, Bilderkennung, Entscheidungen) |
| Internet of Things (IoT) | Vernetzung physischer Geräte (Sensoren, Maschinen) mit dem Internet zur Datenerfassung und -steuerung |
| Cybersecurity / IT-Sicherheit | Schutz von Systemen, Netzwerken und Daten vor Angriffen, Datenverlust und unbefugtem Zugriff |
| Quantencomputing | Nutzung quantenmechanischer Prinzipien für extrem schnelle Berechnungen; noch in der Forschungsphase |
| Blockchain | Dezentrale, manipulationssichere Datenstruktur; bekannt durch Kryptowährungen, aber auch für sichere Transaktionen |
| Augmented/Virtual Reality | Erweiterung oder vollständige Simulation der Realität durch digitale Inhalte (z.B. für Training, Produktvisualisierung) |

## Cloud Computing vs. Edge Computing

| Merkmal | Cloud Computing | Edge Computing |
|---------|----------------|----------------|
| Verarbeitungsort | Zentrales Rechenzentrum | Direkt am Gerät / lokal |
| Latenz | Höher (Daten müssen zum Rechenzentrum) | Sehr niedrig (lokale Verarbeitung) |
| Internetabhängigkeit | Hoch | Gering |
| Typische Anwendung | Web-Apps, Datenspeicherung, SaaS | Autonome Fahrzeuge, Industriesteuerung, IoT |
| Skalierbarkeit | Sehr hoch | Begrenzt durch lokale Hardware |

**Merksatz:** Cloud = zentral, mächtig, aber langsamer. Edge = dezentral, schnell, aber begrenzt.

## Anwendungsfälle für KI

- **Spracherkennung**: Virtuelle Assistenten (Siri, Alexa, Google Assistant)
- **Bilderkennung**: Gesichtserkennung, medizinische Diagnose aus Röntgenbildern
- **Predictive Maintenance**: Vorhersage von Maschinenausfällen anhand von Sensordaten
- **Chatbots / Kundenservice**: Automatisierte Beantwortung von Kundenanfragen
- **Betrugserkennung**: Erkennung ungewöhnlicher Zahlungsmuster in Echtzeit
- **Empfehlungssysteme**: Produktvorschläge bei Amazon, Netflix-Empfehlungen

## Grundprinzipien der IT-Sicherheit (CIA-Triade)

| Prinzip | Begriff | Bedeutung |
|---------|---------|-----------|
| Vertraulichkeit | Confidentiality | Daten dürfen nur von autorisierten Personen eingesehen werden |
| Integrität | Integrity | Daten dürfen nicht unbemerkt verändert werden |
| Verfügbarkeit | Availability | Systeme und Daten müssen für berechtigte Nutzer zugänglich sein |

### Weitere wichtige Sicherheitsprinzipien

- **Authentizität**: Sicherstellung, dass eine Identität echt ist (z.B. durch Passwort, MFA)
- **Nicht-Abstreitbarkeit**: Aktionen können einem Nutzer eindeutig zugewiesen werden (Logging)
- **Least Privilege**: Nutzer erhalten nur die Rechte, die sie für ihre Aufgabe benötigen
- **Defense in Depth**: Mehrere Sicherheitsebenen (kein Single Point of Failure)
